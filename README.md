# mbti 검사
나의 성격과 성향을 알 수 있는 요즘 흔히 접할 수 있는 검사
## 자바스크립트 공부를 위한 mbti 검사 프로젝트
요즘 들어 mbti 검사 사이트가 많아짐에 따라 mbti 검사 사이트를 만들고 싶은 생각이 들어서 이 프로젝트를 계획하게 됨.

|폴더 |내용|
|:---|:---:|
|assets| 폰트,사진,검사 질문지|
